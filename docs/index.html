Complete DBTL Workflow

Five integrated stages from data ingestion to regulatory compliance

📊

Stage 1: Data Layer

Comprehensive organism databases, genomic data loading, and quality validation for 5 industrial strains.

  • E. coli, Yeast, B. subtilis, C. glutamicum, P. putida
  • Automated schema validation
  • Data quality scoring
🧠

Stage 2: AI Pathway Engine

Monte Carlo Tree Search retrosynthesis, enzyme selection, and codon optimization powered by ML simulations.

  • Thermodynamic scoring
  • ESM-2 embedding compatibility
  • Promoter/RBS design
⚗️

Stage 3: Flux Analysis

Flux Balance Analysis (FBA), strain optimization, and toxicity prediction for robust designs.

  • FBA, pFBA, FVA analysis
  • Multi-objective optimization
  • Intermediate toxicity assessment
🔄

Stage 4: DBTL Loop

Simulated Design-Build-Test-Learn cycles with Bayesian optimization and fermentation modeling.

  • 48 constructs per cycle
  • ODE-based fermentation
  • MPC bioreactor control
🏭

Stage 5: Scale-up & Compliance

Production scale-up prediction, downstream processing, and regulatory assessment.

  • 2L → 20,000L cascade
  • Purification train design
  • GRAS & BSL compliance

Quick Start

Get running in minutes with Docker or pip

# Build the image
docker build -t synbio-pipeline .

# Run a complete pipeline
docker run --rm synbio-pipeline \
  python main_pipeline_runner.py \
  --organism ecoli \
  --molecule lycopene \
  --cycles 3 \
  --output-dir /app/output

# Or use docker-compose
docker-compose up pipeline
# Install dependencies
pip install -r requirements.txt

# Run the pipeline
python main_pipeline_runner.py \
  --organism scerevisiae \
  --molecule vanillin \
  --cycles 5

# View results
cat ./pipeline_output/final_report.json
# Clone the repository
git clone https://github.com/your-org/synbio-pipeline.git
cd synbio-pipeline

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Linux/Mac
# or
venv\Scripts\activate     # Windows

# Install dependencies
pip install -r requirements.txt

# Run tests
make test

# Run example
python main_pipeline_runner.py --organism ecoli --molecule lycopene
